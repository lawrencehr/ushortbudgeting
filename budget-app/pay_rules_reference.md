# Pay Calculation Structure Reference

## Artists (Category E) – Television (Non-Program)

### 1. Full-Time & Part-Time

**Monday – Saturday**
- 0 – 7.6h: 1.0x Base
- 7.6 – 9.6h: 1.5x Base
- 9.6h+: 2.0x Base

**Sunday**
- All Hours: 2.0x Base

**Public Holiday**
- All Hours: 2.5x Base

### 2. Casual Employees
*(Includes casual loading)*

**Monday – Saturday**
- 0 – 7.6h: 1.25x Base
- 7.6 – 9.6h: 1.875x Base
- 9.6h+: 2.5x Base

**Sunday**
- All Hours: 2.0x Base

**Public Holiday**
- All Hours: 2.5x Base

## Crew (Excluding Category E)

**Definition of Hours:**
- Ordinary Hours: First 7.6 hours
- Overtime: After 7.6 hours

### 1. Full-Time & Part-Time

**Monday – Friday**
- 0 – 7.6h: 1.0x Base
- 7.6 – 9.6h: 1.5x Base
- 9.6h+: 2.0x Base

**Saturday**
- 0 – 7.6h: 1.5x Base (Ordinary Penalty)
- 7.6 – 9.6h: 1.75x Base
- 9.6h+: 2.0x Base

**Sunday**
- 0 – 7.6h: 1.75x Base (Ordinary Penalty)
- 7.6h+: 2.0x Base

**Public Holiday**
- All Hours: 2.5x Base

### 2. Casual Employees
*(Includes casual loading)*

**Monday – Friday**
- 0 – 7.6h: 1.25x Base (Standard Casual Rate)
- 7.6 – 9.6h: 1.875x Base
- 9.6h+: 2.5x Base

**Saturday**
- 0 – 7.6h: 1.75x Base
- 7.6 – 9.6h: 2.1875x Base
- 9.6h+: 2.5x Base

**Sunday**
- 0 – 7.6h: 2.0x Base
- 7.6h+: 2.5x Base

**Public Holiday**
- All Hours: 3.125x Base
